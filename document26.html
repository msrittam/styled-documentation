Practical 1: Identifying Vowels and Consonants

In the first practical exercise, we will create a simple Java program that determines whether a given character is a vowel or a consonant. This program utilizes the Scanner class for user input and a switch statement to check for vowels. Here is the code:

import java.util.Scanner;
public class Practical {
    public static void main(String[] args) {
        int a = 0;
        Scanner Scr = new Scanner(System.in);
        System.out.println("Enter an alphabet:");
        char b = Scr.next().charAt(0);
        switch (b) {
            case 'a':
            case 'e':
            case 'i':
            case 'o':
            case 'u':
            case 'A':
            case 'E':
            case 'I':
            case 'O':
            case 'U':
                a++; }
        if (a == 1) {
            System.out.println("It is a vowel");
        } else if ((b >= 'a' && b <= 'z') || (b >= 'A' && b <= 'Z')) {
            System.out.println("It is a consonant");
        } else { System.out.println("Wrong input"); } } }

In this code, we first initialize a variable a to count the vowels. We then read a character from the user. The switch statement checks if the character is a vowel. If it is, we increment a. Finally, we determine if the character is a vowel, consonant, or an invalid input.

Practical 2: Calculating Factorial

In the second practical, we will compute the factorial of a number entered by the user. The factorial of a number is the product of all positive integers up to that number. Here’s the code for this practical:

import java.util.Scanner;
public class Practical2 {
    public static void main(String[] args) {
        int i = 1, c = 1;
        Scanner Scr = new Scanner(System.in);
        System.out.println("Enter the number: ");
        int a = Scr.nextInt();
        while (i <= a) {
            c = c * i;
            i++; }
        System.out.println("Factorial of entered number is: " + c); } }

This program uses a while loop to calculate the factorial by multiplying all integers from 1 to the input number a.

Practical 3: Armstrong Numbers

An Armstrong number is a number that is equal to the sum of its own digits each raised to the power of the number of digits. The following code checks if the entered number is an Armstrong number:

import java.util.Scanner;
public class Practical3 {
    public static void main(String[] args) {
        Scanner Scr = new Scanner(System.in);
        System.out.println("Enter the number:");
        int a = Scr.nextInt();
        int i = 0, b, c;
        c = a;
        while (a > 0) {
            b = a % 10;
            a = a / 10;
            i = i + (b * b * b); }
        if (c == i) {
            System.out.println("It is an Armstrong number.");
        } else { System.out.println("It is not an Armstrong number."); } } }

In this code, we extract each digit, cube it, and add it to a total. If the total matches the original number, it is identified as an Armstrong number.

Practical 4: Identifying Capital Letters

In the fourth practical exercise, we will create a program that identifies and prints all the uppercase letters in a given string. This program iterates over each character of the string:

public class Practical4 {
    public static void main(String[] args) {
        String a = new String("Hello My Name IS Java");
        for (int i = 0; i < a.length(); i++) {
            if (Character.isUpperCase(a.charAt(i))) {
                System.out.println("The capital letters are as follows: " + a.charAt(i)); } } } }

The program uses a for loop and the Character.isUpperCase() method to check and print each capital letter found in the string.

Practical 5: Finding the Second Most Frequent Character

In this practical, we will determine the second most frequent character in a given string. The program uses an array to count occurrences of each character:

public class Practical5 {
    static final int Pro = 300;
    static char java(String a) {
        int[] b = new int[Pro];
        for (int i = 0; i < a.length(); i++)
            (b[a.charAt(i)])++;
        int x = 0, y = 0;
        for (int i = 0; i < Pro; i++) {
            if (b[i] > b[x]) {
                y = x;
                x = i;
            } else if (b[i] > b[y] && b[i] != b[x]) {
                y = i; } }
        return (char) y; }
    public static void main(String[] args) {
        String a = new String("JavaPractical");
        char c = java(a);
        if (c != '\0') {
            System.out.println("Second most frequent character is: " + c);
        } else { System.out.println("No character found."); } } }

This program maintains a count of characters and determines which character occurs second most frequently.

Practical 6: Counting Odd and Even Numbers

This practical focuses on counting how many numbers in an array are odd and how many are even. Here’s the code:

import java.util.Arrays;
public class Practical6 {
    public static void main(String[] args) {
        int[] elements = {1, 2, 3, 4, 5};
        System.out.println("Numbers: " + Arrays.toString(elements));
        int a = 0;
        for (int i = 0; i < elements.length; i++) {
            if (elements[i] % 2 == 0) { a++; } }
        System.out.println("Odd number count: " + a);
        System.out.println("Even number count: " + (elements.length - a)); } }

In this code, we iterate through the array to count even and odd numbers, displaying the counts at the end.

Practical 7: Exception Handling

This exercise introduces exception handling in Java. The program demonstrates how to catch different types of exceptions:

public class Practical7 {
    public static void main(String[] args) {
        try {
            int a = 7 / 0; // This will cause ArithmeticException
            String b = null;
            System.out.println(b.length()); // This will cause NullPointerException
            int c[] = new int[10];
            c[20] = 30; // This will cause ArrayIndexOutOfBoundsException
        } catch (ArithmeticException e) {
            System.out.println(e);
        } catch (NullPointerException e) {
            System.out.println(e);
        } catch (ArrayIndexOutOfBoundsException e) {
            System.out.println(e); } } }

The program intentionally causes exceptions to demonstrate how they can be caught and handled gracefully.
