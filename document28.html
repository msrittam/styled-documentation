Java with DB2 connection and the query executions

try { Class.forName("com.ibm.db2.jcc.DB2Driver"); }
catch (ClassNotFoundException e2) {
    System.out.println("Please include Classpath Where your DB2 Driver is located");

Following the successful loading of the driver, the code establishes a connection to the database using the following block:

String url = "jdbc:db2://localhost:50000/user";
Connection con = (Connection) DriverManager.getConnection(url, "db2admin", "asdfghjkl");

PreparedStatement ps = con.prepareStatement("Delete from loginpage WHERE username =(?)");

The method executeUpdate() is then called to execute the deletion operation.
